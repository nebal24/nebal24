##  About This Repository

This repository is actually my GitHub profile page.  
GitHub has a feature where if you create a repository with the same name as your username, the README file shows up on your profile.

Everything here is written using Markdown, which is just a simple way to format text.

The parts that look “dynamic” (like the stats and badges) are actually images generated from external tools. These tools connect to GitHub, get some data (like commits or activity), and turn it into visuals.

So even though it looks interactive, it’s really just images that update automatically when the page loads.

I’m using this README as a small portfolio to show what I work with and what I’m interested in.
